RESUMO COMPLETO DO PROJETO — Certificate Generate
Visão Geral
SaaS independente em Ruby on Rails 8.1.3 + SQLite (MVP) para geração automatizada de certificados de cursos e palestras. Com interface web profissional (ERB + Bootstrap 5) e API REST (namespace api/v1) para futura integração com ToolCenter (Hub IAM/SSO com JWT RS256).

Fluxo Completo da Aplicação
Usuário acessa Dashboard → vê estatísticas e ações rápidas
Cria Evento (título, carga horária, período, local, instrutor)
Cadastra Participantes (manual ou importa planilha Excel/CSV)
Na tela do Evento, seleciona participantes → clica "Gerar Certificados"
Sistema gera PDF com QR Code único para cada participante
Envia automaticamente por email (aluno + BCC instrutor)
Registra tudo no EmailLog
Aluno valida o certificado escaneando QR Code (funciona offline)
Link temporário de download como backup
